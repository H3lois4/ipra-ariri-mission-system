# IPRA no Ariri — Mission System

Sistema desenvolvido para apoiar o trabalho missionário da IPRA no Ariri, com foco em:

- Uso em campo (mobile)
- Funcionamento offline (PWA)
- Controle de usuários
- Registro e visualização de atividades
- Transparência e organização dos dados

## Tecnologias
- Django
- Django Rest Framework
- PWA
- SQLite / PostgreSQL

⚠️ Este repositório é público apenas para fins de portfólio.
Dados reais, documentos e informações sensíveis **não são versionados**.
